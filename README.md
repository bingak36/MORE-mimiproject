#연습용 웹사이트 크롤링

HTML및 CSS만을 사용하여 최대한 melon 홈페이지와 유사하도록 작성한 html코드입니다.
java script를 사용한 부분은 최대한 제외하고 일부분은 스포티파이의 ui를 가져왔습니다.

---

CSS를 구성할 땐 header, main, footer를 분류하여 파일 3개로 겹치지 않도록 작성하였으며 이후 실제 실전에 들어갈 시 hader, main, footer를 나누어서 작성할 예정입니다.
---

반응형 ui를 구현하기 위해서 label태그를 input 태그와 동기화하여 header부분의 사용자명과 로그아웃 버튼을 추가하였고 로그아웃 버튼 클릭시 실제 로그아웃이 되는것 처럼 기존에 있던 유저명과 로그아웃 글자를 지웠다.


가장 먼저 문제가 생겼던 부분은 상속입니다.
예상하지 못한 초기화 과정 중에서 백그라운드 색이 포함되거나 백그라운드에 해당하지 않는 부분에서 지속적으로 백그라운드를 최대한 밖으로 꺼내어서 작성하였습니다.

두번째로 생겨난 문제는 java script 없이 java script를 어떻게 구현을 할 수 있을까에 관한 답이였습니다.
다양한 발상을 위해서 최대한 구현하보려 하였으나 일부 부분(무한 스크롤)은 절대로 java script를 거치지 않고는 만들 수 없어 나중을 기약하며 코드 작성을 제외하였지만
인기 있어요나 로그인 버튼 같은 부분에 관해서는 radio button을 사용하여 checked기능을 활용할 수 있는 부분을 확인하여서 사용해보았습니다.

---

처음부터 완전한 설게를 하고 작성한것이 아닌 html의 본문 내용만 먼저 작성 후 header부분부터 작성하다보니 중간 중간에 css코드를 다시 작성해야하는 부분도 생겨났고
가장 어려웠던 부분은 class의 개수는 많아지는데 어떤 이름을 붙여야 되는지 내가 그걸 보고 바로 바로 어디인지 파악할 수 있도록 작성하는것이 제일 어려웠던 부분이였습니다.
